# React + Vite Project

 # Booklet – React Books Library App

A modern and responsive Books Library Web App built using **React.js**, **SCSS**, **Bootstrap**, and **Figma UI** design.

Live Demo:  https://kalpanadatacode.github.io/Booklet-Project/

---

## 📚 Overview

Booklet allows users to:

* Login & Sign up
* Browse available books
* View full book details
* Add/create new books
* Manage personal book collection

This project follows clean UI/UX practices and mobile‑first responsive design.

---

## 🚀 Features

* 🔐 **Authentication** (Login/Signup)
* 📖 **Book Listing** with search & filter
* 📝 **Create / Add New Book**
* 📘 **Book Details Page**
* 💾 **Local Data Storage / API Ready Structure**
* 📱 Fully **responsive design**
* 🎨 UI designed in **Figma**

---

## 🛠️ Tech Stack

* **React.js**
* **SCSS**
* **Bootstrap 5**
* **React Router DOM**
* **Figma** (for UI design)

## ⚙️ Installation & Setup

```bash

git clone <repo-link>

# Navigate to project folder
cd Booklet-Project

# Install dependencies
npm install

# Start the server
npm start
```
